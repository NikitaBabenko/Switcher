# Detailed description — Русский

Набрали **`ghbdtn`** вместо **`привет`**? Нажмите **Ctrl+Shift+L** — и абракадабра превращается в нормальный текст, без перепечатывания, без выхода из поля и без отправки хотя бы одного символа на сервер.

## Чем VibeNest Switcher отличается

- **100% открытый исходный код** — полный код на GitHub: https://github.com/NikitaBabenko/Switcher
- **Полностью офлайн по умолчанию** — нет телеметрии, нет аналитики, нет вызовов в облако; языковая модель встроена в расширение (~270 КБ)
- **Без аккаунта, без логина, без оплаты** — установил и работай
- **12 языков из коробки** — большинство аналогов покрывают один-два
- **Manifest V3** — минимум разрешений, готов к политике Chrome 2025
- **Кросс-браузер** — работает в Chrome и Chromium-форках; Edge и Firefox порты в планах

## Привычно тем, кто работал с Punto

Если вы пользовались Punto Switcher на Windows — это та же идея, но внутри браузера, с открытым кодом и без отправки текста за пределы вашего устройства. Работает в любом текстовом поле в открытом вебе, не только в чатах.

## Что умеет

- **Горячая клавиша** (Ctrl+Shift+L) и кнопка в popup'е — мгновенное исправление активного поля
- **Режим «вставить и исправить»** — вставьте кривой текст в popup, получите правильный
- **Правый клик по выделению** в контекстном меню
- **Автоисправление по ходу набора** в стиле Punto Switcher — опционально; Backspace отменяет
- **Отмена per-page** — откатывает последнее исправление на текущей странице одной кнопкой
- **Адаптеры для сайтов**: Twitter/X, Facebook, Messenger, VK, Instagram, Telegram Web, WhatsApp Web, Discord, Slack, Reddit, LinkedIn, Twitch, Mastodon + универсальный адаптер для всего остального
- **Политика сайтов** — блокируйте конкретные хосты (например, банк) или разрешайте только из whitelist'а
- **Умные дефолты по языкам** — на первой установке расширение само определит языки набора по локали браузера; вы можете изменить это в любой момент

## Как это работает

Детектор переставляет каждый символ по всем парам выбранных раскладок, затем символьная trigram-модель, натренированная на частотных словарях, оценивает оба направления и выбирает то, что выглядит естественно. Эвристики Caps Lock и «натуральности регистра» подтягивают пограничные случаи. Всё это работает в вашем браузере.

## Кому пригодится

- Билингвам, которые переключаются между английским и другим алфавитом по нескольку раз в день
- Переводчикам, набирающим то на исходном, то на целевом языке прямо посреди абзаца
- Разработчикам и айтишникам в мультиязычных окружениях
- Хелпдеск-командам, которые не хотят отправлять коллеге письмо «Уважаемые коллеги, ifkjvtt…»

## Приватность

По умолчанию расширение работает оффлайн. Встроенный детектор живёт в вашем браузере; ни текст, ни метаданные никуда не уходят. Разрешения минимальны, и каждое объяснено в политике конфиденциальности: https://github.com/NikitaBabenko/Switcher/blob/main/extension/PRIVACY.md

Опционально можно включить fallback на удалённый API (по умолчанию выключен), если хотите подключить свой эндпоинт. Базовая установка не делает сетевых запросов вовсе.
